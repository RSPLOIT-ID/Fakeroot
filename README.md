# Informasi
**Fakeroot** <br>
Fakeroot Adalah Tools Untuk Memberikan Akses Root Di Termux Kita

# Install
How To Install It :
```
$ apt update && apt upgrade -y
$ pkg install git -y
$ git clone https://github.com/RSPLOIT-ID/Fakeroot
$ cd Fakeroot
$ bash exec.sh
```
# Perintah
How To Use :
```
$ sudo
```
How To Uninstall Termux Root :
```
$ sudo clear
```
